# Lottie Animasyon Dosyaları

Bu klasöre yüz doğrulama animasyonları eklenmelidir:

- `face_front.json` - Ön yüz animasyonu
- `face_left.json` - Sol yüz animasyonu  
- `face_right.json` - Sağ yüz animasyonu

## Nereden Bulunur?

Lottie animasyonları şu siteden ücretsiz indirilebilir:
- https://lottiefiles.com/

Örnek arama terimleri:
- "face scan"
- "face detection"
- "face recognition"
- "biometric"

## Not

Animasyonlar yoksa uygulama icon'lar ile çalışacaktır. Animasyonlar opsiyoneldir.
